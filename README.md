# 2-Dimensional Kinetics Simulator

Requirements:
1. npm/node
2. docker

## Usage

build
```sh
npm run build
```

start
```sh
npm start
```

open
```sh
npm run open
```

stop
```sh
npm stop
```

## Test

setup
```sh
npm run test-setup
```

test
```
npm test
```

cleanup
```sh
npm run test-cleanup
```

dev mode testing
```
npm run test:dev
```

## Maintenance

lint
```sh
npm run lint
```

lint fix
```sh
npm run lint:fix
```

clean
```sh
npm run clean
```
