# 2-Dimensional Kinetics Simulator

Requirements:
1. npm/node
2. docker

## Usage

build
```shell script
npm run build
```

start
```shell script
npm start
```

open
```shell script
npm run open
```

stop
```shell script
npm stop
```

## Test

setup
```shell script
npm run test-setup
```

test
```shell script
npm test
```

cleanup
```shell script
npm run test-cleanup
```

dev mode testing
```shell script
npm run test:dev
```

## Maintenance

lint
```shell script
npm run lint
```

lint fix
```shell script
npm run lint:fix
```

clean
```shell script
npm run clean
```
